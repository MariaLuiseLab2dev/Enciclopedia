Enciclopédia HTML

Elementos HTML

<a>
O elemento HTML <a> (ou elemento âncora), com seu atributo href, cria um hiperlink para páginas da Web, arquivos, endereços de e-mail, locais na mesma página ou qualquer outra coisa que um URL possa endereçar. O conteúdo de cada um deve indicar o destino do link.

Atributos principais

href — A URL para a qual o hiperlink aponta.
Exemplos de esquemas de URL:
  • Números de telefone com URL's tel:
  • Endereços de e-mail com URL's mailto:
  • Mensagens de texto SMS com URL's sms:
Outros recursos de URL podem localizar partes específicas:
  • Seções de uma página com fragmentos de documento
  • Partes de texto específicas com fragmentos de texto
  • Partes de arquivos de mídia com fragmentos de mídia
target — Onde exibir o URL vinculado:
  • _blank: nova aba ou janela
  • _parent: contexto de navegação pai
  • _top: contexto de navegação mais alto
  • _unfencedTop: usado para atravessar além da raiz do quadro cercado

Quando usar:

Quando quiser criar um link para outro site, página interna, e-mail ou número de telefone.

  <p>Este é um parágrafo com um <a href="https://www.google.com">link para o Google</a>.</p>
<blockquote>
Indica que o texto delimitado é uma citação estendida. Normalmente, isso é renderizado visualmente por recuo.

Atributos principais

cite
Uma URL que designa um documento ou mensagem de origem para as informações citadas. Este atributo destina-se a apontar para informações que explicam o contexto ou a referência para a citação.

Quando usar:

Quando quiser criar uma citação.

<blockquote cite="https://www.huxley.net/bnw/four.html"> <p>         Words can be like X-rays, if you use them         properly—they'll go through anything. You         read and yo're pierced.
      </p>
  </blockquote>
<p>
Representa um parágrafo. Os parágrafos geralmente são representados em mídia visual como blocos de texto separados de blocos adjacentes por linhas em branco e/ou recuo de primeira linha, mas os parágrafos HTML podem ser qualquer agrupamento estrutural de conteúdo relacionado, como imagens ou campos de formulário.

Atributos principais

style
Contém declarações de estilo CSS a serem aplicadas ao elemento. Observe que é recomendado que os estilos sejam definidos em um arquivo ou arquivos separados. Esse atributo e o elemento <style> têm principalmente a finalidade de permitir um estilo rápido, por exemplo, para fins de teste.
id
Define um identificador (ID) que deve ser exclusivo em todo o documento. O objetivo do atributo ID é identificar um único elemento ao vincular (usando um identificador de fragmento), script ou estilo (com CSS).
class
Uma lista separada por espaços das classes do elemento. As classes permitem que CSS e JavaScript selecionem e acessem elementos específicos por meio dos seletores de classe ou funções como o método Document.getElementsByClassName().

Quando usar:

Quando quiser criar um link para outro site, página interna, e-mail ou número de telefone.

  <p>Este é um parágrafo .</p>
<img>
O elemento HTML <img> representa uma imagem incorporada ao documento. Ele não possui tag de fechamento e deve indicar a fonte da imagem através do atributo src.

Atributos principais

src
URL da imagem a ser exibida.
alt
Texto alternativo que descreve a imagem para leitores de tela ou quando a imagem não carrega.
width
Largura da imagem (pixels).
height
Altura da imagem (pixels).

Quando usar:

Quando quiser inserir uma imagem fixa ou decorativa em seu conteúdo.

  <img src="paisagem.jpg" alt="Montanhas ao amanhecer" width="600" height="400">
<ul>
O elemento HTML <ul> cria uma lista não ordenada. Seus itens são definidos com <li> e geralmente exibidos com marcadores.

Atributos principais

type
(legado) Define o estilo do marcador (círculo, quadrado, etc.).
compact
(obsoleto) Indicava apresentação mais compacta da lista.

Quando usar:

Quando quiser agrupar itens em uma lista sem impor uma ordem numérica.

  <ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
  </ul>
<li>
O elemento HTML <li> define um item dentro de listas ordenadas (<ol>), não ordenadas (<ul>) ou de descrição (<menu>).

Atributos principais

value
(em ol) Define o valor inicial de numeração para o item.
type
(em ol legados) Define o estilo de numeração.

Quando usar:

Dentro de <ul>, <ol> ou <menu> para listar cada elemento.

  <ol>
    <li value="5">Item inicial no 5</li>
    <li>Item 6</li>
  </ol>
<button>
O elemento HTML <button> cria um botão clicável, que pode disparar ações em formulários ou scripts.

Atributos principais

type
Define o comportamento: button (padrão), submit (envia formulário), reset (limpa formulário).
disabled
Quando presente, desabilita o botão.
name
Nome do controle de formulário.
value
Valor enviado com o formulário.

Quando usar:

Para acionar ações, enviar ou resetar formulários, ou disparar scripts.

  <button type="submit">Enviar</button>
<form>
O elemento HTML <form> representa uma seção de documento que contém controles interativos para enviar dados ao servidor.

Atributos principais

action
URL para onde os dados serão enviados.
method
Método HTTP: get ou post.
enctype
Tipo de codificação de dados (ex.: multipart/form-data para upload de arquivos).
target
Onde exibir a resposta (_self, _blank, etc.).

Quando usar:

Ao precisar coletar e enviar dados do usuário via HTTP.

  <form action="/submit" method="post">
    <input type="text" name="nome">
    <button type="submit">Enviar</button>
  </form>
<input>
O elemento HTML <input> representa um controle de dados, como campo de texto, checkbox, rádio, entre outros.

Atributos principais

type
Tipo de controle (text, password, checkbox, radio, email, etc.).
name
Nome do controle no envio do formulário.
value
Valor inicial ou selecionado.
placeholder
Texto de dica quando vazio.
required
Indica que o campo é obrigatório.

Quando usar:

Para coletar dados dos usuários em formulários.

  <input type="email" name="email" placeholder="seu@exemplo.com" required>
<section>
O elemento HTML <section> representa uma seção genérica de conteúdo, normalmente com um título próprio, usada para agrupar temas relacionados.

Atributos principais

id
Identificador único para navegação ou estilos.
class
Classes para estilização ou seleção via script.

Quando usar:

Para dividir a página em blocos temáticos, cada um com seu próprio título.

  <section id="sobre">
    <h3>Sobre Nós</h3>
    <p>Texto descritivo...</p>
  </section>

Herança CSS

O que é herança em CSS?

Herança é um mecanismo onde certas propriedades CSS, quando aplicadas a um elemento pai, são passadas para seus elementos filhos.

O que é propriedade?

Propriedade se refere a um dos atributos que definem o estilo de um elemento HTML — como color, font-size, border.

Cada definição de propriedade do CSS diz se tal propriedade é herdada por padrão ("Inherited": sim) ou é não herdada por padrão ("Inherited": não). Isso controla o que ocorre quando nenhum valor é especificado para uma propriedade em um elemento.

Propriedades Herdadas Por Padrão

Quando nenhum valor de uma propriedade herdada foi especificado em um elemento, o elemento obtém o valor computado desta propriedade em seu elemento pai.

Como font-family, text-align, entre outros.

Exemplo:

 p {
  color: green;
 }

E o markup:

  <p> Este é um parágrafo <em>enfatizado</em> dentro dele.</p>

As palavras "texto enfatizado" vão aparecer em verde, já que o elemento em herdou o valor da propriedade color do elemento p. Ele não obtém o valor inicial da propriedade (o qual é a cor usada pelo elemento raiz quando a página não especifica nenhuma cor).

Nota: A palavra-chave inherit permite aos autores especificar explicitamente a herança. Ela funciona tanto em propriedades herdadas como não herdadas.

Propriedades não-herdadas

Quando nenhum valor para uma propriedade não herdada é especificado em um elemento, o elemento obtém o valor inicial desta propriedade.

Como margin, padding, border, entre outros.

Exemplo:

Um exemplo típico de uma propriedade não herdada é a propriedade border. Dadas as seguintes regras de estilo:

 p {
  border: medium solid;
 }

E o markup:

  <p> Este é um parágrafo <em> enfatizado </em> dentro dele.</p>

As palavras "texto enfatizado" não vão ter uma borda (já que o valor inicial de border-style é none).

Sinalizador !important

O que é?

É uma diretiva que ao ser adicionada ao final de uma declaração de propriedade, ele eleva essa declaração ao topo da cascata faznedo com que anule qualquer outra declaração para a mesma propriedade.

Exemplo:

Você utiliza quando quiser deixar aquela propriedade acima de qualquer outra, ma o uso é perigoso e deve ser evitado.

 p {
  color: red !important;
 }

Seu resultado vai ser um parágrafo sempre vermelho independe se vier outra declaração colocando roxo, por exemplo.
